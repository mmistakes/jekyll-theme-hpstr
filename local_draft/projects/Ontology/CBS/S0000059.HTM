System Architect Symbol details for "Lifelong Property"
Acquired Permanent Property (Class)
Description: "Subject Property", effective from a point in time, which cannot be changed.
Acquired Temporary Property (Class)
Description: "Subject Property" which exists until it is observed to change.
Inherent Temporary Property (Class)
Description: "Subject Property" which is applicable from the start of existence of the "Subject" but may cease to be so before the end of the "Subject"'s existence.
Examples:
Virginity
Non-smoking
6th finger on one hand
Two kidneys.

Lifelong Property (Class)
Description: "Inter Subject Relationship" which is effective from the start of the "Subject" which it describes and will not change
SPr/Persistence (Generalization Node)
Base Class: "Subject Property"
Definitions Used:
Subject Property (Class)
Subject Property (Class)
Description: Condition or state associated with a "Subject".
Transient Property (Class)
Description: "Subject Property" applying only at the time of its "Observing Act".
