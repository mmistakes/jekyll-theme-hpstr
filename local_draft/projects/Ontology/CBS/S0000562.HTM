System Architect Symbol details for "Text Valued Characteristic"
Category Valued Characteristic (Class)
Description: "Subject Characteristic" with a value defined by a "Value Category".
Comment: See "Value Category" for typical values.
Image Valued Characteristic (Class)
Description: "Subject Characteristic" having a value represented by an image.
Comment:
The image may be video or audio and may be static or moving. Examples would include:
Scans
X rays
Videos of colposcopies etc
Recordings of sounds or conversations.



Definitions Used:
Image (Class Attribute)
Measured Characteristic (Class)
Description: "Subject Characteristic" with a declared value defined as a number.
Comment: An example would be "75 bpm" as a pulse rate.


Definitions Used:
numeric value (Class Attribute)

Subject Characteristic (Class)
Description: "Subject Property" which is a specific condition, state or other item of information observed about a "Subject" at a point in time and excluding its location, identifiers and relationships to other "Subjects".
Comment:
Derived in part from CBS v2:

Any specific condition, state, property or other item of information about a "Subject" at a point in time.

Text Valued Characteristic (Class)
Description: "Subject Characteristic" having a value expressed by a text string.
Comment: The string may be structured or unstructured. Examples include pathology and radiology reports, unstructured assessments as well as amplification and addenda to phenomena. For example, an informal assessment may be recorded as a Text Valued Characteristic. The obvious drawback is that any automated analysis of these is difficult if not impossible.


Definitions Used:
Text (Class Attribute)
Timepoint Valued Characteristic (Class)
Description: "Subject Characteristic" whose value is specified by a "Timepoint"
Comment: For example: DoB.
