System Architect Symbol details for "Direct Activity For Subject"
Activity For Subject (Class)
Description: "Activity" directed towards a single "Subject".
Comment:
Previous definitions from which the above have been distilled:

An act which changes either the state, or the knowledge of the state, of a "Subject" (CBS v2).


AFS/Scope (Generalization Node)
Base Class: "Activity For Subject"
Definitions Used:
Activity For Subject (Class)
Context Activity (Class)
Description: "Activity" acting as a context for one or more other "Activities".
Comment:
The context activity may be generated before the component activities exist.
Common types of Context Activity are:
Consultant Episode
Ward Stay
Consultation
Out Patient Attendance
Case Conference.

Note that Context Activities may themselves be components of other Context Activities.

Direct Activity For Subject (Class)
Description: "Activity For Subject" not acting as the context for any other.
