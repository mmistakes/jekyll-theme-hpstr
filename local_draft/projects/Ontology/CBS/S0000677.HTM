System Architect Symbol details for "Category Valued Qualifier"
Category Valued Qualifier (Class)
Description: "Phenomenon Qualifier" with a value defined by a "Value Category".
Comment: See "Value Category" for typical values.
Image Valued Qualifier (Class)
Description: "Phenomenon Qualifier" having a value represented by an image.
Comment:
The image may be video or audio and may be static or moving. Examples would include:
Scans
X rays
Videos of colposcopies etc
Recordings of sounds or conversations.



Definitions Used:
Image (Class Attribute)
Measured Qualifier (Class)
Description: "Phenomenon Qualifier" with a declared value defined as a number.
Comment: An example would be "25%" as the extent of burn damage.


Definitions Used:
numeric value (Class Attribute)
Phenomenon Qualifier (Class)
Description: Extension or modification of a "Subject Phenomenon" or another "Phenomenon Qualifier".
PhenQ/Value Type (Generalization Node)
Base Class: "Phenomenon Qualifier"
Text Valued Qualifier (Class)
Description: "Phenomenon Qualifier" having a value expressed by a text string.
Comment: The string may be structured or unstructured. Examples include pathology and radiology reports, unstructured assessments as well as amplification and addenda to phenomena or qualifiers with other values. For example, an Image Valued Qualifier containing an X-ray image may be qualified by a report from the radiologist.


Definitions Used:
Text (Class Attribute)
Timepoint Valued Qualifier (Class)
Description: "Phenomenon Qualifier" whose value is specified by a "Timepoint"
Comment: For example: date from which the severity of a condition changed.
value (Association)

Value Category (Class)
Description: "Category" which may be used to give a value to a "Category Valued PhenorQual".
Comment:
Typical examples are:
"True"
"False"
"Left" (laterality)
"Right" (ditto)
"High"
"Low".
Obviously, there are sets from which selection may be made. This should be governed by constraining selection to one out of a set of species categories within a single Classification Element.
