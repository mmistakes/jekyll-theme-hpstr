Ordered Classification Element Defining Diagram
"Classification Element" whose species have an ordered relation one to another.
Examples:
The Glasgow coma scale.
Attributes:
Duplicates allowed (M) (Boolean)
If "True", duplicate values of "Order" in the associated "Ordered Species Relations" are allowed.
Associations From:
Only One genus Many Ordered Species Relation
Supertypes:
Classification Element
Ordered Species Relation Defining Diagram
"Species Relation" which has an ordinal relation with others in the same "Ordered Classification Element"
Examples:
CIN1, CIN2, CIN3 etc used in cytology; staging of diseases.
Attributes:
Order (M) (Integer)
Position held by an "Ordered Species Relation" within an "Ordered Classification Element". Note that duplicates are not illegal.
Associations To:
Many genus Only One Ordered Classification Element
Supertypes:
Species Relation
