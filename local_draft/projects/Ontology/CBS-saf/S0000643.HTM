System Architect Symbol details for "Activity Type Offered"
AAT/essential type (Generalization Node)
Base Class: "Agent's Activity Type"
Definitions Used:
Agent's Activity Type (Class)
Activity Type Offered (Class)
Description:
An "Activity Type" that an "Agent" makes a general offer to perform.


Comment: There is no need for any authorisation for this type of offer - it can be offered by a "Free Agent". Example would be setting up a business to supply goods that are not regulated by legislation (i.e. a pharmacists shop would not be covered by this type.)
Agent's Activity Type (Class)
Description: A means of establishing that an "Agent" may perform, offer to perform or authorise others to perform certain "Activity Types".
Comment: For examples see subtypes.
Authorised Agent Activity (Class)
Description: "Activity Type for Agent" that is subject to some type of authorisation or external control.
Comment: Such control can be interdictions, permissions and exceptions (including "Activity State Types") to one or many Activity Types.
