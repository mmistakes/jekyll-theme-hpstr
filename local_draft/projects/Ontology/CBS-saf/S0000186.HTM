Established Focus (Class)

FFA/State (Generalization Node)

Focus For Further Activity (Class)
Focus For Further Activity (Class)
Description: "Property Qualifier" being an "Agent's" view of a "Subject Property" which may have attention directed towards it.
Rejected Focus (Class)
Description: "Focus For Further Activity" which was initially decided to be a "Relevant Focus" but which is not now so considered.
Comment: Previously known as "Discarded" but name changed to make it consistent.
Relevant Focus (Class)
Description: "Focus For Further Activity" indicating that there is an intention to change or address a "Subject Property".
Comment:
Old definition was "Subject Property" for which there is an intention to change. Changed this to be in line with other states.
Suggest change name to "Focus Under Consideration".
